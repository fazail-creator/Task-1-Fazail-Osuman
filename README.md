
Overview

This project is a responsive portfolio website developed as part of my internship at DecodeLabs. The website demonstrates frontend web development skills using HTML5, CSS, and JavaScript.

The main objective of the project is to create a modern, responsive, and user-friendly website that works seamlessly across mobile phones, tablets, and desktop devices.

 Features

* Responsive design
* Mobile-friendly navigation menu
* Hero section
* About section
* Skills showcase
* Contact section
* Smooth scrolling navigation
* Active navigation highlighting

Technologies Used

* HTML5
* CSS
* JavaScript

 Project Structure

project-folder/
│
├── index.html

How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in a browser or use Live Server.
4. The website should load automatically.
